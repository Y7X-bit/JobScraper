## ✨ Features
- 🧠 **Resume Matching** using spaCy NLP
- 📊 **Visualizations** (job counts by location)
- 📍 **Location + Keyword Filters**
- 📁 **Exports** to CSV, Excel, JSON & SQLite
- 🖤 **AMOLED UI** with Red accents
- 🧊 Smooth button hovers & clean UI layout
